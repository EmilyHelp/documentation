Admin Commands Documentation

Command Description Usage
/ban <reason> Bans a user from the group by replying to their message with a specified reason. /ban Spamming the group
/mute <reason> Mutes a user, preventing them from sending messages, by replying with a reason. /mute Inappropriate language
/unmute <reason> Unmutes a previously muted user by replying with a reason. /unmute Completed timeout period
/unban <reason> Unbans a previously banned user by replying with a reason. /unban Appeal approved
/tmute 1h <reason> Temporarily mutes a user for a specified duration (1s, 1m, 1h, 1d) with a reason. /tmute 1h Disruptive behavior
/tban 1h <reason> Temporarily bans a user for a specified duration (1s, 1m, 1h, 1d) with a reason. /tban 1d Repeated rule violations
/kick Kicks a user from the group (they can rejoin) by replying to their message. /kick
/promote <title> Promotes a member to an admin role, optionally assigning a custom title. /promote Moderator or /promote
/demote Demotes an admin to a regular member by replying to their message. /demote
/pin Pins a message to the top of the group chat by replying to it. /pin
/unpin Unpins a previously pinned message by replying to it. /unpin
/botpin Bot sends a message and automatically pins it to the group chat. /botpin
/adminlist Displays the list of current admins in the group. /adminlist
/disableadmin Disables all admin commands for members, admins, and owners until re-enabled. /disableadmin
