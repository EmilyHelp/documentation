Antispam Documentation

Command Description Usage
/antispam Checks the current antispam settings for the group. /antispam
/setantispam 7 10s Sets antispam to delete messages if a member sends 7 messages within 10 seconds. Timer options: 1s, 1m, 1h. /setantispam 7 10s
/disableAntispam Disables the antispam feature for the group. /disableAntispam
/punisha Sets punishment for spammers (kick, mute, tmute, ban). Default: delete message. /punisha kick
