Approve Members Documentation

Command Description Usage
/approve Approves a user by replying to their message. Approved users are exempt from locks, languages, etc., but not spam. /approve
/disapprove Disapproves a previously approved user by replying to their message. /disapprove
/myapprove Checks if the user is approved. /myapprove
/listapprove Lists all approved members in the group. /listapprove
