Command Description Usage
/clkcry Checks if crypto message deletion is enabled. /clkcry
/enablecry Enables deletion of crypto-related messages. /enablecry
/disablecry Disables deletion of crypto-related messages. /disablecry
/punishcry ban/kick/mute/tmute 1h Sets punishment for sending crypto messages (ban, kick, mute, tmute 1h). Default: delete message. /punishcry mute
