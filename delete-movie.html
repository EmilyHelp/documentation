Command Description Usage
/clkmo Checks if movie message deletion is enabled. /clkmo
/enablemov Enables deletion of movie-related messages. /enablemov
/disablemov Disables deletion of movie-related messages. /disablemov
/punishmov ban/mute/kick/tmute 1h Sets punishment for sending movie messages (ban, mute, kick, tmute 1h). Default: delete message. /punishmov ban
