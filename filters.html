Filters Documentation

Command Description Usage
/filter Sets the bot to reply with "reply" whenever "trigger" is said. Use quotes for multi-word triggers. /filter hello Hi there! or /filter "good morning" Good day!
/filters Lists all active filters in the chat. /filters
/stop Stops the bot from replying to the specified trigger. /stop hello
/stopall Stops all filters in the current chat (cannot be undone). /stopall
