Language Blocker Documentation

Command Description Usage
/onlyenglish Deletes all messages that are not in English. /onlyenglish
/onlyrussian Deletes all messages that are not in Russian. /onlyrussian
/onlyhindi Deletes all messages that are not in Hindi. /onlyhindi
/onlyarab Deletes all messages that are not in Arabic. /onlyarab
/langblock Shows which language restriction is currently enabled. /langblock
/disablelang Disables the specified language restriction. /disablelang english
/punishl Sets punishment for messages in restricted languages (kick, mute, tmute, ban). Default: delete message. /punishl mute
