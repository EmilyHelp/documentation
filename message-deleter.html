Message Deleter Documentation

Command Description Usage
/delete link Deletes all messages containing links. /delete link
/delete contact Deletes all messages containing contact information. /delete contact
/delete photo Deletes all messages containing photos. /delete photo
/delete video Deletes all messages containing videos. /delete video
/delete sticker Deletes all messages containing stickers. /delete sticker
/delete gif Deletes all messages containing GIFs. /delete gif
/delete bot Deletes all messages sent by bots. /delete bot
/delete bluetext Deletes all messages with blue text (clickable text). /delete bluetext
/delete commands Deletes all messages containing commands. /delete commands
/delete username Deletes all messages containing usernames. /delete username
/delete emoji Deletes all messages containing emojis. /delete emoji
/delete forwards Deletes all forwarded messages. /delete forwards
/delete font Deletes all messages with custom fonts. /delete font
/delete inline Deletes all messages with inline bot content. /delete inline
/delete numbers Deletes all messages containing numbers. /delete numbers
/delete documents Deletes all messages containing documents. /delete documents
/delete email Deletes all messages containing email addresses. /delete email
/delete voice Deletes all messages containing voice messages. /delete voice
/punishm Sets punishment for deleted message types (kick, mute, tmute, ban). Default: delete message. /punishm ban
