Other Commands Documentation

Command Description Usage
/disable edit Deletes any edited messages in the group. /disable edit
/lockgroup 13-23 Locks the group during the specified time (13:00-23:00, railway/Indian time), preventing messages. /lockgroup 13-23
/report or @admin Reports a message to admins. /report or @admin
/admin off/on Disables or enables admin reports. Default: on. /admin off or /admin on
/log Connects a log channel to receive group activity logs. /log -1001234567890
/nolog Disables the log channel, stopping log messages. /nolog
