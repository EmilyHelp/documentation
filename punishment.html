Punishment Documentation

Command Description Usage
/mywarn Allows a user to check their current warnings. /mywarn
/warn Warns a user, with an optional reason. User is banned after 3 warnings by default (admin/owner only). /warn Spamming or /warn
/unwarn Removes a warning from a user, with an optional reason (admin/owner only). /unwarn False warning or /unwarn
/punishwarn kick/mute/ban/tmute 1hr Sets punishment for users reaching the warn limit (kick, mute, ban, tmute 1hr). Default: ban. /punishwarn mute
/warncount 5 Sets the number of warnings before punishment (default: 3). /warncount 5
