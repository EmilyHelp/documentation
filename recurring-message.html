Recurring Message Documentation

Command Description Usage
/addmsg 1s,1m,1h,1d Sends a message repeatedly at the specified interval (1s, 1m, 1h, 1d) when tagged and replied to. /addmsg 1h
/chkmsg Shows all recurring messages set in the chat. /chkmsg
/clearmsg Clears all recurring messages in the chat. /clearmsg
