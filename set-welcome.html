Set Welcome Documentation

Command Description Usage
/welcome Displays the currently set welcome message. /welcome
/setwelcome Sets a welcome message by replying to a message. Supports markdown: {firstname}, {groupname}, {profilelink}, {username}, <b>bold</b>, <i>italic</i>. Add inline buttons with <button>name url</button>. /setwelcome (reply to a message like Welcome {firstname} to {groupname}! <b>Enjoy</b> our <i>community</i>! <button>Visit google.com</button>)
/removewelcome Disables the welcome message. /removewelcome
